# SpringBootApplication


SpringBootApplication is a sample spring boot project.

### Tech

### Installation

Install the dependencies and devDependencies and start the server.

```sh
JDK 1.7 or later
Maven 3 or later
spring-boot 1.3.5.RELEASE
```
